# 🚀 Ubuntu Wayland Session Saver

A GNOME extension designed to save and restore your application layout across different static workspaces on Ubuntu (Wayland).

## Features
* **Save Session:** Capture all currently open applications and their respective workspaces.
* **Restore Session:** Reopen your apps exactly where you left them.
* **Wayland Native:** Built specifically to handle the security constraints of the Wayland compositor.
